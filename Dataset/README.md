## Dataset Description

This folder contains the raw sales dataset used for the project.

### File:
- **superstore_1000_rows.xlsx**

### Details:
- Contains 1000 sales records
- Superstore-style business data
- Used as the base input for analysis
- Data is cleaned and transformed in the Analysis stage

### Note:
This file represents the **source data** only.  
All calculations, KPIs, and dashboards are created separately.
